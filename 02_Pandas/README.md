# 02_Pandas – Data Analysis with Pandas

This folder contains my structured learning and practical exercises for **Pandas**, the Python library used for data manipulation and analysis.

Goal:  
To confidently load, clean, transform, analyze, and export real-world datasets for Machine Learning and Data Science tasks.
